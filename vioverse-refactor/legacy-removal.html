Migration Notes
  • The old view toggle with icons has been replaced by the new InDesign-style toggle at position x=70px, y=76px
  • The new toggle uses a sliding orange indicator instead of individual button states
  • Icon usage (Lucide icons) has been removed in favor of text-only labels
  • The new toggle follows the Space Grotesk Bold 16pt typography specification
  • All functionality has been preserved in the new implementation
