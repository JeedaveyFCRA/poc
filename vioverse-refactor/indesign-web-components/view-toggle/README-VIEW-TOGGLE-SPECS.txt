This button toggles 3 views

In the toggle OFF state all text should be:
• Font: Space Grotesk Bold, 16pt  
• All text is lowercase
• 30% tint of #253541 (hex #9ba1a6); Do not use rgba transparency

In the toggle ON state all text should be:
• Font: Space Grotesk Bold, 16pt  
• All text is lowercase
• text is white
  
Here are the text positions for OFF and ON states:
"viotagger" x80; y84;w75;h17
"report" x167; y84;w75;h17
"evidence" x255; y84;w75;h17
